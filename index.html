The Pretty GS1 Barcode Scanner

Available now!

Scan GS1 Barcodes

The App supports all common barcode types, but furthermore can understand the GS1 Standard and show all values that are inside of the barcode.

History

Keep track of your history and revisit your barcodes later.

Share

Share the total raw barcode with any other App or person, or copy the values out of the barcode

Supported barcodes:
  • Code 39
  • Code 93
  • Code 128
  • EAN-8
  • EAN-13
  • Data Matrix
  • UPC-E
  • Aztec
  • PDF417
  • ITF14
  • Interleaved 2 of 5 codes
Support GS1 Application Identifiers:
  • Serial Shipping Container Code
  • GTIN
  • GTIN Of Contained Trade Items
  • Lot Number
  • Production Date
  • Due Date
  • Packaging Date
  • Best Before Date
  • Expiration Date
  • Product Variant
  • Serial Number
  • Secondary Data Fields
  • Count Of Items
  • Number Of Units Contained
  • Product Weight In KG
  • Lot Number N
  • Additional Product Identification
  • Customer Part Number
  • Made To Order Variation Number
  • Secondary Serial Number
  • Reference To Source Entity
